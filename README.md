# Context Compressor

Compress context efficiently without losing data

## Features

- Zero external dependencies (stdlib only)
- Easy-to-use CLI interface
- Professional Python implementation
- MIT licensed

## Installation

```bash
pip install -e .
```

Or clone and install:

```bash
git clone https://github.com/Viprasol-Tech/context-compressor
cd context-compressor
pip install -e .
```

## Usage

### Python

```python
from context_compressor import ContextCompressor

result = ContextCompressor.process("data")
print(result)
```

### CLI

```bash
python -m context_compressor "your input here"
```
